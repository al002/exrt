# Exrt

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'exrt'
```

And then execute:

    $ bundle

Or just:

    $ gem install exrt

## Usage

check [exrt-cli](https://github.com/al002/exrt-cli/) for usage example.
